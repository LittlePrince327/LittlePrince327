# STT Whisper Large v3 Turbo 검증 요약

> 회사 원본 음성·정답 전사·개인정보·내부 경로·원본 코드는 공개하지 않습니다.

## 현재 단계

- 현재 CER: 후처리 포함 최저 평균 CER `0.0761`
- 현재 확인된 범위: 992개 평가 샘플의 Whisper 기본 모델 6종 비교, Full fine-tuning Grid Search, 세그먼트 병합·텍스트 정규화 후처리
- 추후 과제:
  - 동일 평가셋 CER·WER 재측정
  - 모델 원본 성능과 후처리 효과 분리
  - LoRA와 Full fine-tuning 동일 조건 비교
  - 운영 환경 연동 및 배포 검토

## 검증 기준

| 구분 | 조건 | 평균 CER |
| --- | --- | ---: |
| 기본 모델 비교 | Whisper 6종, 992개 평가 샘플 | Large v3 Turbo `0.0889` |
| Full fine-tuning 모델 원본 | lr `3.16e-6`, 4 epochs, linear scheduler, 16kHz | `0.1756` |
| 후처리 포함 결과 | lr `1e-5`, 3 epochs, linear scheduler, 16kHz, 세그먼트 병합·텍스트 정규화 | `0.0761` |

## 데이터와 설정

- 기반 모델: `openai/whisper-large-v3-turbo`
- 입력 표준화: 16kHz
- 세그먼트 길이: 최대 30초
- 학습 활용 데이터: 텍스트가 있는 4,727개 세그먼트, 971개 rx/tx 화자 채널 음성
- 평가 데이터: 992개 음성 샘플
- LoRA 실험: `r=16`, `alpha=32`, `dropout=0.05`, 대상 모듈 `q_proj`, `v_proj`
- LoRA CER: 미측정

## 담당 업무

- 음성 데이터 정제와 16kHz 포맷 표준화
- 최대 30초 세그먼트 구성
- Whisper 모델군 비교와 Full fine-tuning Grid Search
- CER 기반 오류 분석
- 세그먼트 병합·반복어·특수문자·공백·텍스트 정규화 후처리 구현
- LoRA/PEFT 경량 튜닝 실험

## 표현 원칙

- `0.0889`는 기본 모델군 비교 결과이며 파인튜닝 최종 성능으로 표현하지 않습니다.
- `0.1756`은 모델 원본 출력 기준이며 `0.0761`은 후처리 포함 결과입니다.
- 일부 행의 최솟값을 파일 전체 평균으로 사용하지 않습니다.
- 운영 환경 연동 및 배포는 추후 과제입니다.
