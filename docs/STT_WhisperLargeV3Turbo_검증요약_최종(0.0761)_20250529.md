# STT Whisper Large v3 Turbo 검증 요약

> 회사 원본 음성·정답 전사·개인정보·내부 경로·원본 코드는 공개하지 않습니다.

## 현재 단계

- 현재 CER: 후처리 포함 최저 평균 CER `0.0761`
- 현재 확인된 범위: 993건 규모의 수동 청취·묵음 지점 분할과 입력 사양 확인, 평가 가능한 992개 샘플의 Whisper 기본 모델 6종 비교, Full fine-tuning Grid Search, 세그먼트 병합·텍스트 정규화 후처리
- 추후 과제:
  - 자동 묵음 구간 분할과 수동 분할 결과 비교
  - 화자 분리 기반 CER 개선 효과 검증
  - 밴드패스 필터 기반 노이즈 전처리 효과 검증
  - 운영 환경 연동 및 배포

## 검증 기준

| 구분 | 조건 | 평균 CER |
| --- | --- | ---: |
| 기본 모델 비교 | Whisper 6종, 992개 평가 샘플 | Large v3 Turbo `0.0889` |
| Full fine-tuning 모델 원본 | lr `3.16e-6`, 4 epochs, linear scheduler, 16kHz | `0.1756` |
| 후처리 포함 결과 | lr `1e-5`, 3 epochs, linear scheduler, 16kHz, 세그먼트 병합·텍스트 정규화 | `0.0761` |

## 데이터와 입력 처리

- 기반 모델: `openai/whisper-large-v3-turbo`
- 입력 사양 확인: PCM/WAV의 샘플레이트·채널·샘플 폭 확인
- 변환 형식: 16kHz·mono·16-bit WAV
- 코덱 종류: 2~3개 수준이며 코덱별 별도 평가지표는 만들지 않음
- 수동 분할: 993건 규모의 음성을 직접 청취하고, 30초를 넘는 파일은 중간 묵음 지점에서 나누어 각 구간을 30초 이하로 구성
- 분할 이유: 묵음 구간에서 이상한 특수문자와 앞서 나온 문장이 반복되는 현상 완화
- 학습 활용 데이터: 텍스트가 있는 4,727개 세그먼트, 971개 rx/tx 화자 채널 음성
- 평가 데이터: 평가 가능한 992개 음성 샘플
- 고오류 재추론: CER 0.3 이상인 인식 품질이 낮은 22개를 선별해 학습된 여러 모델로 재추론·비교
- LoRA 실험: `r=16`, `alpha=32`, `dropout=0.05`, 대상 모듈 `q_proj`, `v_proj`
- 노이즈 전처리 실험: 400~3200Hz 밴드패스 필터 코드가 있으나 CER 개선 효과는 미검증

## 담당 업무

- 샘플레이트·채널·샘플 폭 확인과 16kHz·mono·16-bit WAV 변환
- 993건 규모 음성 수동 청취와 묵음 지점 기준 최대 30초 분할
- Whisper 모델군 비교와 Full fine-tuning Grid Search
- CER 기반 검증과 고오류 샘플 재추론
- 세그먼트 병합·반복어·특수문자·공백·텍스트 정규화 후처리 구현
- LoRA/PEFT 경량 튜닝 실험
- 밴드패스 필터 기반 노이즈 전처리 후보 실험

## 표현 원칙

- `0.0889`는 기본 모델군 비교 결과이며 파인튜닝 최종 성능으로 표현하지 않습니다.
- `0.1756`은 모델 원본 출력 기준이며 `0.0761`은 후처리 포함 결과입니다.
- 993건 규모의 수동 청취·분할과 992개 평가 샘플을 구분합니다.
- 자동 묵음 구간 분할, 화자 분리, 밴드패스 필터 효과 검증, 운영 환경 연동 및 배포는 추후 과제입니다.
- 일부 행의 최솟값을 파일 전체 평균으로 사용하지 않습니다.
