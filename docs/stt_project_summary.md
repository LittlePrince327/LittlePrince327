# 콜센터 STT 프로젝트 검증 요약

> 이 문서는 공개 가능한 범위의 프로젝트 요약입니다. 회사 원본 음성·정답 전사·개인정보·내부 경로·원본 코드는 공개하지 않습니다.

## 프로젝트 상태

- 기간: 2025.03–2025.05
- 범위: 콜센터 음성 전처리, Whisper 모델 비교, 도메인 파인튜닝, CER 기반 평가, 후처리 파이프라인 실험
- 배포 여부: **실제 운영 환경 배포 없음**
- 성격: 실험·검증 및 내부 분석 단계

## 검증된 내용

- 입력 음성을 16kHz로 표준화하고 최대 30초 단위로 세그먼트 구성
- Whisper 6종을 992개 음성 샘플에서 비교
- 기본 모델군 비교에서 Whisper Large v3 Turbo 평균 CER 8.89% 확인
- 학습률·에포크 조합을 자동 반복 비교
- 모델 원본 출력 기준 최적 설정: `learning_rate=3.16e-6`, `4 epochs`, `linear scheduler`, `16kHz`
- 세그먼트 병합·텍스트 정규화 등 후처리 포함 최저 CER 설정: `learning_rate=1e-5`, `3 epochs`, `linear scheduler`, `16kHz`
- LoRA/PEFT 경량 튜닝을 별도 실험으로 수행

## 해석 원칙

1. `CER 8.89%`는 Whisper 기본 모델군 비교 결과이며 파인튜닝 최종 성능으로 직접 표현하지 않습니다.
2. 모델 원본 CER와 후처리 포함 CER를 구분합니다.
3. LoRA 실험 모델을 최종 선정 모델로 표현하지 않습니다.
4. 실제 운영 환경에 배포했다고 표현하지 않습니다.

## 담당 업무

- 음성 데이터 정제 및 포맷 표준화
- 최대 30초 세그먼트 구성
- Whisper 모델 비교 및 파인튜닝
- 학습 설정 자동 반복 실험
- CER 기반 오류 분석
- 세그먼트 병합·텍스트 정규화 파이프라인 구현

## 공개 제한

프로젝트는 회사 내부 데이터를 사용했기 때문에 원본 데이터와 코드는 공개하지 않습니다. 이 저장소에는 채용 검토를 위한 고수준 설명과 검증된 수치만 기록합니다.
