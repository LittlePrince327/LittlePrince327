# 콜센터 STT 프로젝트 검증 요약 (0.0761)

> 이 문서는 공개 가능한 범위의 프로젝트 요약입니다. 회사 원본 음성·정답 전사·개인정보·내부 경로·원본 코드는 공개하지 않습니다.

## 현재 단계

- 기간: 2025.03–2025.05
- 범위: 콜센터 음성 전처리, Whisper 모델 비교, 도메인 파인튜닝, CER 기반 평가, 후처리 파이프라인 실험
- 현재 단계: 실험·검증 완료
- 추후 과제: 독립 평가셋 재검증, WER 병행 측정, 운영 환경 연동 및 배포

## 검증된 내용

- 입력 음성을 16kHz로 표준화하고 최대 30초 단위로 세그먼트 구성
- Whisper 6종을 992개 음성 샘플에서 비교
- 기본 모델군 비교에서 Whisper Large v3 Turbo 평균 CER `0.0889` 확인
- 학습률·에포크 조합을 자동 반복 비교
- 모델 원본 출력 기준 최적 설정: `learning_rate=3.16e-6`, `4 epochs`, `linear scheduler`, `16kHz`, 평균 CER `0.1756`
- 세그먼트 병합·텍스트 정규화 등 후처리 포함 최저 설정: `learning_rate=1e-5`, `3 epochs`, `linear scheduler`, `16kHz`, 평균 CER `0.0761`
- LoRA/PEFT 경량 튜닝을 별도 실험으로 수행

## 추후 과제

1. 독립 평가셋에서 CER·WER 재검증
2. 모델 원본 성능과 후처리 개선 효과 분리
3. LoRA와 Full fine-tuning의 동일 조건 비교
4. 운영 환경 연동 및 배포

## 담당 업무

- 음성 데이터 정제 및 포맷 표준화
- 최대 30초 세그먼트 구성
